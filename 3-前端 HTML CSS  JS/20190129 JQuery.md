# 1 JQuery 概述

## 1.1  使用 JQuery 的原因

   1. 选择器多
   2. 无DOM操作繁杂
   3. 浏览器兼容性好
   4. 动画效果好

##  1.2. JQuery  的定义

> 对 JavaScript 代码的封装

```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>Test</title>
		<script type="text/javascript" src="js/js.js" charset="UTF-8"></script>
		<script>
			function Test(){
				// 通过 Js 的 DOM 方法获得 div 节点对象
				console.log(document.getElementById("div_1"));
				// 通过 模拟 JQuery 封装方式 获得 div 节点对象
				console.log($("div_1"));
			}
			
			// 获得节点对象为 null 必须放在 函数中 执行 
			console.log(document.getElementById("div_1"));
			console.log($("div_1"));
			
		</script>
	</head>
	<body>
			<div id="div_1"></div>
			<input type="button" name="test" id="test" value="测试"  onclick="Test()"/>
	</body>
</html>

```

```js
/****js.js 文件*************/
function $(id){
	
	var dom = document.getElementById(id)
	return dom;
	
}

```



## 1.3 JQuery 中 $ 的含义

1. **作用1： $(function)**

> 相当于 window.onload=function(){}，但是功能比前者更强。
>
> 前者只加载最后一个函数，JQuery 加载全部函数

- 第一种方式

```js
$(function(){

});
```

- 第二种方式

```html
jQuery(function(){

});
```

- 第三种方式

```js
 $(document).ready(function(){	
	
});
```

2. **作用2: $(selector)** 

> 选择器

```js
 $(function(){
    $("tr:odd").css("background-color","yellow");
})
```

# 2 JQuery 选择器

> 代替 DOM 对象，使获得节点的方法变得更加多种多样

## 2.1 基本选择器

### 2.1.1  ID 选择器


```js
//获得 div 节点中的所有 span 节点
```

### 2.2.2  后代选择器

```js
//获得 div 节点中的是所有 直系 span 节点， 不包括 p 节点下的 span 的节点
$("div >span").css("background-color","red");
```

```js
// 获得 id为span_1 节点后的 系挨着的节点 span。 注意：必须是系挨着的
```

```js
// 获得 id 为 span_1 节点后的 所有直系  同类 span 节点
```

## 2.3  基本选择器和子元素选择器

### 2.3.1 基本选择器

> 将相同父元素中所有子元素归为一个父元素中

#### 2.3.1.1 first

```js
//获得 ul 节点下第一个 li 节点
$("ul li:first").css("background-color","red");				
$("ul li").first().css("background-color","red");
```

#### 2.3.2.2  last

```js
//获得 ul 节点下最后一个 li 节点      注意： 3个ul都看做成一个 ul，li 都归属于同一个 ul 节点
$("ul li:last").css("background-color","red");
$("ul li").last().css("background-color","royalblue");
```

#### 2.3.2.3 eq(n)

> 索引从 0 开始 ( n=0,1,2...)

```js
//根据索引获取节点,注意 索引从 0 开始
$("ul li:eq(4)").css("background-color","green");
$("ul li").eq(5).css("background-color","greenyellow");
```

#### 2.3.2.4  odd   even

```js
//根据 奇偶 获得
$("ul li:odd").css("background-color","red");   //选择奇数列
$("ul li:even").css("background-color","blue"); //选择偶数列
```

#### 2.3.2.5  gt lt

```js
 //根据大于 4 选择
//$("ul li:gt(4)").css("background-color","yellow");
//根据小于 4 选择 
//$("ul li:lt(4)").css("background-color","yellow");
```



### 2.3.2 子元素选择器

> 区分每个相同标签的父元素对应的子元素

#### 2.3.2.1  first-child

```js
/***********子选择器************/

```



#### 2.3.2.2  last-child

```js
//选择每个 ul 节点的最后一个 li 节点的值
$("ul li:last-child").css("background-color","green");
```



#### 2.3.2.3  nth-child(n) 

> n 从 1 开始

```js
//根据 li 的位置查找,从 1 开始
$("ul li:nth-child(3)").css("background-color","greenyellow");
```



#### 2.3.2.4  only-child


## 10.3 闭包原理

> 好处：
>
>  1、可以减少全局变量的对象，防止全局变量过去庞大，导致难以维护  <br />
>
>  2、防止可修改变量，因为内部的变量外部是无法访问的，并且也不可修改的。安全<br />
>
>  3、读取函数内部的变量，另一个就是让这些变量的值始终保持在内存中。

```js
	/**** 闭包原理**********/
		
				
$(function(){

    $("input").click(function(){

        alert("测试");
        var a  = 10;

        var test = (function(){

            alert(a);
            var b = 20; 
            return b;   //体现闭包，节省内存空间，安全

        });

        alert("b="+test());
    });

```

## 11.4 总结

1. 选择器的选择
   1. 单选框、文本框、按钮可以选择用 ID 选择器，也是最常用的方式
   2. 多选框有多个可以选择使用 属性 Attitude 选择器与表单属性对象选择器混用
   3. 节点的值可以使用`parent()` 、`childrend()` 、`eq() `方法

2. `each() `方法是对节点集合的遍历，相比使用 `for `更加便捷

3. 不可将 JQuery 对象与 JS相混

   注意在节点使用事件触发函数传入`this` 指针时，用 JQuery 操作需要将 JS 对象转为 JQuery对象用`$(th)`
